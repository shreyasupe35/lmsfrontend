#lms fronted

##setup instruction
1.clone the project
...
    git clone https://github.com/shreyasupe35/lmsfrontend.git
...

2.Move into the director
...
    cd  "lms2 frontend"

...

3.Install dependencies
...
    npm i
...

4.Run the server


    npm run dev
...
